## 🛠️ Tech Stack & Engineering Decisions

To ensure maximum runtime reliability and eliminate environment-breaking bugs under a compressed timeframe, we chose a unified, zero-config full-stack architecture:

* **Frontend:** React / Next.js (App Router) styled with **Tailwind CSS** and customized visual layers for an immersive, high-fidelity dark-mode interface.
* **Backend:** Next.js Serverless Route Handlers (`/api/*`) executing instantly on Vercel's global edge network.
* **Real AI Services:** Integrated live via direct secure API hooks connecting to **Google Gemini Flash API / Hugging Face Inference Serverless API** (No simulated/mock responses).
* **Version Control & CI/CD:** Maintained via GitHub with automated, atomic pipeline deployment triggers on Vercel.
